# Fruits-Game
very basic web game that was made as a final project for the Serious Game Fundementals course
it was made using basic javascript and generic and game-based code files
the idea was to make an educational game that helps children develop a specific learning skill/ knowledge 
in this game the goal was to help kids distinguish between healthy and unhealthy food
healthy food gives points and increases score
unhealthy makes you lose points and slow speed, after certain mistakes the game ends
